```

│
├── 📁 styles/
│   └── main-styles.css .................. עיצוב עמוד הבית
│
├── 📁 student/

│   └── styles.css ....................... עיצוב אזור תלמיד
│
├── 📁 teacher/
│   ├── index.html ....................... ממשק ניהול
│   └── styles.css ....................... עיצוב אזור מורה
│
├── 📁 games/
│   └── digital-culture/
│       ├── index.html ................... המשחק

```

---

## 🚀 התחלה מהירה

### 3 צעדים:

1. **פתח את `index.html` או `landing.html`**
2. **בחר אזור תלמיד או מורה**
3. **התחל ללמוד!**

### עם שרת מקומי:
```bash
python -m http.server 8000
# פתח: http://localhost:8000
```
